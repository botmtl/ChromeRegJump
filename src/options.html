by

You're almost done! There are 2 things required to make this extension work:

  1. Download Sysinternals RegJump from Microsoft, and extract it to: 

    %LOCALAPPDATA%\Google\Chrome\User Data\Default\Extensions\\_0\host\regjump

  2. Register the host program to communicate with RegJump, by running register-host.bat located in: 

    %LOCALAPPDATA%\Google\Chrome\User Data\Default\Extensions\\_0\host
When you're done, press the Verify button to make sure everything is up and running!
